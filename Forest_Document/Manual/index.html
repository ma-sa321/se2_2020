インストール方法

1. Forest(GitHub)にアクセスし、codeをクリックする。

その後、urlをコピーする。またはzipファイルをDLする。

2.任意のディレクトリに移動する。

$ cd {任意のディレクトリ}

3-1. cloneする場合、以下のコマンドを実行する。

$ git clone https://github.com/ma-sa321/se2_2020.git

3-2. zipを解凍する場合、以下のコマンドを実行する。

$ unzip se2_2020.zip

4. 展開したディレクトリへ移動する。

$ cd se2_2020

5. 中身の確認。

$ ls -l
